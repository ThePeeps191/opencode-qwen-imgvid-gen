# Opencode + Qwen Image/Video Generation Tools

OpenCode tools for generating images and videos with Wan and HappyHorse through the [Qwen Token Plan](https://www.qwencloud.com/pricing/token-plan).

- Similar to how GPT models in Codex are able to use GPT Image 2 for automatic image generation, these tools allow models within OpenCode to do the same (though video generation is also supported!) provided a Qwen/Alibaba Token Plan is connected to Opencode.
- Models used are HappyHorse 1.1 and Wan 2.7 Image by defualt (the best available models).
